## Информация про виртуальную валюту и контент

Контент существует двух видов:

1. Базовый

```bash
  Доступен всем пользователям по базовой подписке.
```

2. Продвинутый

```bash
  Доступен бесплатно лишь по продвинутой подписке, при подписке уровня "базовый" скрыт, показ доступен лишь при покупке через токены.
```

Токен:

Вирутальная валюта получаемая раз в неделю на обоих уровнях подписки (базовый - 10, продвинутый - 70), также возможна покупка валюты за реальные деньги.
Используется для показа недоступного контента, а также для доната креаторам.
Цена токена при покупке состовляет приблизительно 1.15 рубля, цена при выводе состовляет 0.5 рублей.
